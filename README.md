# SQL Data Warehouse Project

This repository contains my personal SQL Data Warehouse project, where I am applying the Medallion architecture (Bronze, Silver, Gold) and building the project step by step as I learn.

## 🟫 Bronze Layer
This layer stores the raw data exactly as received, without any cleaning or transformations.

## 🥈 Silver Layer
This layer contains cleaned and standardized data.  
It includes:
- Data type corrections
- Duplicate removal
- Handling missing values
- Basic transformations

## 🥇 Gold Layer
This is the business-ready analytical layer.  
It includes:
- Fact tables
- Dimension tables
- KPIs and metrics
- Aggregated data for reporting

## Tools Used
- SQL Server
- GitHub
- Medallion Architecture
